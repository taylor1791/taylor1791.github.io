URL Redirect Test Page

This page renders any URLs passed via query params as links and can optionally redirect to the first URL after a delay.

Query Parameters

  • ?url=https://example.com — Adds a link to the page. Repeatable (e.g. ?url=A&url=B).
  • ?redirect=3000 — Redirects to the first url after the given number of milliseconds.
